# OverTheWire Bandit Notes

This repository contains my personal notes and writeups for the OverTheWire Bandit wargame.

I am using Bandit to build stronger Linux command-line fundamentals, file navigation habits, and basic CTF problem-solving skills. Each level includes my actual thought process, the commands I used, mistakes I ran into, and what I learned from solving it.

The goal of these notes is not to make a perfect polished walkthrough, but to document how I worked through each challenge while improving my understanding of Linux, permissions, files, encoding, searching, filtering, and command chaining.

## Current Progress

- Bandit 0–10 completed
- More levels will be added as I continue through the wargame

## Topics Covered So Far

- SSH basics
- Reading files with `cat`
- Handling filenames with spaces and special characters
- Hidden files and directories
- File type identification with `file`
- Searching with `find`
- Filtering text with `grep`
- Sorting and finding unique lines with `sort` and `uniq`
- Extracting readable strings with `strings`
- Decoding Base64 with `base64 -d`

## Purpose

This repo is mainly for tracking my learning and showing my progression through hands-on Linux and CTF practice.
